Article Generation Schedule

New articles are automatically generated every Monday at 9:00 UTC. Articles will only be generated if there isn't already an article for that week. The scheduler can be controlled using the manage_scheduler.py script: 

# Enable automatic article generation
python scripts/manage_scheduler.py enable

# Disable automatic article generation
python scripts/manage_scheduler.py disable
